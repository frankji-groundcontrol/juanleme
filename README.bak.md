## ✨ 核心功能亮点

### 1. 🗺️ 沉浸式学习路线图 (Roadmap)
告别枯燥的文字教程。我们将复杂的开发流程拆解成一个个小任务卡片。你只需要像玩游戏闯关一样，点亮一个个节点，不知不觉就完成了整个项目。
*   **状态可视化**：灰色的还没开始，蓝色的正在进行，绿色的已经搞定！一眼看清进度。
*   **任务拆解**：从“灵感构思”到“Demo 上线”，每一步都有清晰指引。

### 2. ✍️ 智能协作编辑器
这不仅仅是一个写字的地方，更是一个会思考的笔记本。
*   **所见即所得**：支持 Markdown 格式，图片、链接随意插，排版精美。
*   **AI 灵感侧边栏**：写不出文案？描述你的需求，AI 帮你润色、扩写，甚至提供代码思路。一键采纳，效率翻倍。

### 3. 📱 全平台无缝体验
无论你习惯用电脑、iPad 还是手机，体验都一样丝滑。
*   **移动优先**：在地铁上也能用手机随时记录灵感。
*   **触控友好**：大大的按钮，流畅的滑动，手指操作也精准。

### 4. 📤 一键导出成果
辛辛苦苦写好的方案，一键生成精美的 PDF 或 Markdown 文档。直接拿去展示，或者发给队友，专业范儿十足。

## 🚀 快速上手指南

### 对于参与者 (Participants)
1.  **注册账号**：花 10 秒钟，填个邮箱和昵称，开启你的旅程。
2.  **加入工作坊**：输入组织者给你的 **6位邀请码**，瞬间进入活动现场。
3.  **开始闯关**：点击 Roadmap 上的第一个卡片，按照引导回答问题。
4.  **求助 AI**：遇到难题，点击右侧的机器人图标，和它聊聊你的困惑。
5.  **提交作业**：完成所有节点后，导出你的专属项目文档。

### 对于组织者 (Organizers)
1.  **创建舞台**：新建一个工作坊，设置好酷炫的封面和简介。
2.  **设计关卡**：在后台拖拽编辑 Roadmap，定制适合你学员的学习路径。
3.  **邀请学员**：生成专属邀请码，分享给现场的小伙伴。
4.  **掌控全场**：实时查看大家的进度，发现谁掉队了，及时给予帮助。

## ❓ 常见问题 (FAQ)

**Q: 我完全不懂编程，能用吗？**
A: **太能了！** 这就是专门为你设计的。我们用最通俗的语言引导你，配合 AI 助手，你会发现“立项”和“产品设计”其实很有趣。

**Q: 这是个手机 App 还是网站？**
A: 它是一个 **Web App**（网页应用）。你不需要下载安装包，打开浏览器就能用。不管是电脑还是手机，访问我们的网址即可。

**Q: 我的数据安全吗？**
A: 非常安全。你的所有回答和创意都加密存储，只有你和授权的管理员能看到。
